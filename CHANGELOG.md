# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-07-06
### Added
- **Multi-Agent Simulation Engine**: Central Coordinator Agent orchestrating 6 specialized sub-agents.
- **9-View Single Page Application**: Home, Dashboard, Incident Center, Resources, Volunteers, Communications, Reports, Analytics, Settings.
- **5 Emergency Scenarios**: Flood, Earthquake, Wildfire, Hurricane, Extreme Heat.
- **Material Design UI**: Responsive layouts, dark/light mode, and smooth micro-animations.
- **Python Backend**: FastAPI implementation for static file serving and potential future API expansion.
- **Dockerization**: Dockerfile included for easy deployment to cloud platforms.
- **CI/CD**: GitHub Actions workflow for automated testing and docker builds.
